## 1. 密码验证规则修改

### 问题
- 密码只能包含字母和数字，不能包含特殊字符（如 `.`、`_`、`*` 等）
- 用户需要使用包含特殊字符的密码（如 `Smg.2025`）

### 修改文件

**后端 DTO 文件：**
- `oc-platform-user/src/main/java/com/ocplatform/user/dto/RegisterRequest.java`
- `oc-platform-user/src/main/java/com/ocplatform/user/dto/ChangePasswordRequest.java`
- `oc-platform-user/src/main/java/com/ocplatform/user/dto/ResetPasswordRequest.java`

**修改内容：**
```java
// 旧正则
@Pattern(regexp = "^[a-zA-Z0-9]+$", message = "密码只能包含字母和数字")

// 新正则
@Pattern(regexp = "^[a-zA-Z0-9!@#$%^&*()_+\\-=\\[\\]{};':\"\\\\|,.<>\\/?]+$", message = "密码只能包含字母、数字和常见特殊字符，不能包含空格")
```
