# 2026-03-07 全面代码审查与优化

## 工作概述

对 OC Platform 项目进行了全面的代码审查，涵盖前端代码、后端代码、数据库设计和容器配置，发现并修复了多个安全性、健壮性和代码质量问题。

## 审查发现的问题

### 前端问题

#### 严重
1. **内存泄漏风险** - `setInterval` 未在组件卸载时清理
   - 影响文件: `Login/index.tsx`, `Register/index.tsx`, `Profile/index.tsx`, `ForgotPassword/index.tsx`
   - 修复: 创建 `useCountdown` Hook 统一管理倒计时逻辑

2. **竞态条件风险** - `AdminLayout` 使用 `setTimeout` 延迟检查权限
   - 影响文件: `AdminLayout.tsx`
   - 修复: 移除 `setTimeout`，使用 `useRef` 防止重复检查

#### 中等
1. **错误处理不完整** - 多处 `catch` 块使用 `/* handled */` 注释
   - 修复: 添加 `console.error` 记录错误

2. **console.log 残留** - 生产代码中残留调试日志
   - 影响文件: `ProductEdit.tsx`, `ProductDetail/index.tsx`, `FeedbackSection.tsx`
   - 修复: 移除调试日志

### 后端问题

#### 严重
1. **JWT Token 黑名单未验证** - 用户登出后 Token 仍然有效
   - 影响文件: `JwtAuthenticationFilter.java`
   - 修复: 添加黑名单检查逻辑

2. **登录限流锁定时间不一致** - 设置1分钟但提示15分钟
   - 影响文件: `AuthService.java`
   - 修复: 统一为15分钟

3. **用户状态检查不完整** - 未检查 LOCKED 状态
   - 影响文件: `AuthService.java`
   - 修复: 添加 LOCKED 状态检查

4. **日志中打印验证码** - 验证码明文记录到日志
   - 影响文件: `EmailService.java`
   - 修复: 移除验证码明文输出

#### 中等
1. **请求参数边界验证缺失** - 分页参数无最大值限制
   - 影响文件: `ProductController.java`
   - 修复: 添加 `@Max(100)` 验证

2. **输入参数未 trim** - 登录账号未去除首尾空格
   - 影响文件: `AuthService.java`
   - 修复: 添加 `.trim()` 处理

### 数据库问题

#### 严重
1. **索引缺失** - 多个表缺少必要索引
   - 修复: 添加 `idx_role_permissions_role_id`, `idx_email_verify_expires`, `idx_categories_parent` 等索引

2. **触发器缺失** - `user_oauth_bindings` 和 `site_feedbacks` 表缺少 `updated_at` 触发器
   - 修复: 添加触发器

3. **时间戳类型不一致** - `captcha_records.created_at` 使用 `TIMESTAMP` 而非 `TIMESTAMPTZ`
   - 修复: 统一为 `TIMESTAMPTZ`

4. **is_latest 字段唯一约束缺失** - 可能导致同一产品多个最新版本
   - 修复: 添加部分唯一索引

### 容器配置问题

#### 严重
1. **敏感信息硬编码** - 配置文件中包含真实凭据
   - 影响文件: `application.yml`, `docker-compose.dev.yml`
   - 修复: 移除真实凭据，使用环境变量

2. **网络暴露风险** - 开发环境端口绑定到 0.0.0.0
   - 影响文件: `docker-compose.dev.yml`
   - 修复: 绑定到 127.0.0.1

#### 中等
1. **nginx.conf 缺少安全头** - 缺少 CSP 和 HSTS
   - 影响文件: `nginx.conf`
   - 修复: 添加 `Content-Security-Policy`, `Strict-Transport-Security`, `Permissions-Policy`

2. **日志配置缺失** - 未配置日志轮转
   - 影响文件: `docker-compose.yml`
   - 修复: 添加日志配置
