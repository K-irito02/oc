## 问题分析

### 根本原因

1. **数据库约束问题**: `product_versions_platform_check` 只允许 WINDOWS, LINUX, MACOS，但前端有 IOS, ANDROID 选项
2. **重复提交问题**: 版本创建按钮没有loading状态，用户快速点击会导致重复提交

### 解决方案

1. 更新数据库约束支持所有平台类型
2. 添加 `creatingVersion` 状态和按钮loading效果
3. 添加发布状态验证确保产品发布时有已发布版本
