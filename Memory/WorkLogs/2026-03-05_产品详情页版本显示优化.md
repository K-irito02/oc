# 产品详情页版本显示优化

> 完成时间: 2026-03-05

## 完成的功能

### 1. 版本发布说明展示
- 产品详情页版本列表增加发布说明展示（支持中英文切换）
- 版本发布时自动更新产品的 `latestVersion` 字段
- 产品详情页侧边栏"版本"信息实时显示最新已发布版本号

### 2. 展示版本多选功能（按平台+架构）
- 数据库新增 `display_versions` JSONB 字段存储平台+架构映射
- 格式：`{"WINDOWS_x64": 123, "MACOS_arm64": 456}`
- 每个平台+架构组合可以独立设置展示版本
- 管理后台版本管理中添加"设为展示"按钮

### 3. 按钮延迟优化
- "显示在详情页" Switch 添加 loading 状态
- "设为展示" Button 添加 loading 状态
- 防止用户重复点击

## 技术要点

1. **JSONB 存储**: 使用 PostgreSQL JSONB 存储平台+架构到版本ID的映射
2. **TypeHandler**: 自定义 MyBatis TypeHandler 处理 Map<String, Long> 类型
3. **Loading 状态**: 使用 React useState 管理按钮 loading 状态，防止重复点击
4. **国际化**: 所有新增文案支持中英文双语
