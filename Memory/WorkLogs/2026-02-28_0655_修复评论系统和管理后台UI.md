## 问题修复详情

### 问题 1: 评论计数显示为 0

**现象**: 产品详情页评论标签显示 "评论 (0)"，但实际有评论

**原因**: MyBatis-Plus 分页插件未启用，导致 `selectPage` 方法返回的 `total` 始终为 0

**修复**: 
```java
// MybatisPlusConfig.java
interceptor.addInnerInterceptor(new PaginationInnerInterceptor(DbType.POSTGRE_SQL));
```

### 问题 2: 回复评论时"未知错误"

**现象**: 回复评论后显示"未知错误"，评论未保存

**原因**: PostgreSQL `inet` 类型与 Java `String` 类型不兼容

**修复**:
```sql
ALTER TABLE product_comments ALTER COLUMN ip_address TYPE varchar(45);
```

### 问题 3: 嵌套评论显示

**现象**: 回复显示为独立评论，未嵌套在父评论下方

**修复**: 重新设计 `CommentItem` 组件，支持：
- 递归渲染回复
- 显示被回复人
- 展开/收起更多回复
- 视觉层级区分

### 问题 4 & 5: 管理后台 UI 对齐

**修复**: 统一组件高度和间距
- Header 和 Sidebar Logo 区域：`h-14`
- 搜索框和筛选框：`h-9`
