# 技能和MCP使用规则

## 1. 可用技能列表

### MCP
- **shadcn ui**: 用于前端UI组件的快速构建
- **Sequential Thinking**: 用于复杂问题的顺序思考和分析
- **Tavily search**: 用于网络搜索和信息获取
- **context7**: 用于上下文理解和处理
- **GitHub**: 用于GitHub相关操作和代码管理
- **Playwright**: 用于前端测试和自动化

### SKILLS
- **skill-creator**: 用于创建新技能
- **fix**: 用于代码修复和格式化
- **frontend-design**: 用于前端设计和UI美化
- **ui-ux-pro-max**: 用于专业UI/UX设计和设计系统生成
- **vite-cache-components**: 用于Vite项目的缓存组件实现
- **webapp-testing**: 用于Web应用测试
- **oc-platform-manager**: 用于oc平台项目管理，支持容器、后端和前端状态日志记录
- **pr-creator**: 用于创建Pull Request
- **git-commit-workflow**: 用于Git提交工作流管理
- **find-skills**: 用于发现和安装技能
- **code-reviewer**: 用于代码审查
